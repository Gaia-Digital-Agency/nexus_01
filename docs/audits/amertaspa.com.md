# Technical Audit — amertaspa.com
**Server:** hostinger · **Audited:** 2026-06-11 · **Status:** live

## Verified signals
- HTTP/redirects: Site redirects from non-HTTPS to HTTPS (`https://www.amertaspa.com/`) via an HTTP/2 301 redirect, handled by WordPress.
- robots.txt: `robots.txt` file is present, containing `User-agent: *` and `Crawl-Delay: 20`.
- sitemap: `sitemap_index.xml` is present, generated by Rank Math SEO Plugin, and references `https://www.amertaspa.com/page-sitemap.xml`. `sitemap.xml` returns an HTTP 404.
- HTTPS/headers: HTTPS is enforced. `content-security-policy: upgrade-insecure-requests` header is present. `x-powered-by: PHP/8.2.30` header is present. `server: hcdn` indicates Hostinger CDN is active. `alt-svc: h3=":443"` indicates HTTP/3 support.
- Platform/version: CMS is WordPress 7.0. PHP version is 8.2.30. Hosting platform is Hostinger with hpanel.
- Plugins/theme/security: Rank Math SEO Plugin is identified. WordPress REST API (`wp-json`) endpoints are exposed. `content-security-policy: upgrade-insecure-requests` header is present.
- Schema presence: JSON-LD schema is present on the homepage.
- Speed/CWV signals: Upstream response times (`x-hcdn-upstream-rt`) observed between 0.254s and 1.943s.
