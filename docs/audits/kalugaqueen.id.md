# Technical Audit — kalugaqueen.id

> **In plain terms (for the team):** Good news: your site is secure (HTTPS, Content Security Policy) and Google can fully crawl it (robots.txt, sitemap, 301 redirects). Performance is also well-optimized with HTTP/2 and HTTP/3 support and active LiteSpeed caching. **Server:** hostinger · **Audited:** 2026-06-11 · **Status:** live

## Verified signals
- HTTP/redirects: The site redirects from an unspecified initial URL to `https://www.kalugaqueen.id/` via a single HTTP/2 301 redirect, resolving to a 200 OK status.
- robots.txt: `robots.txt` is present and allows all user agents (`User-agent: *`, `Disallow:` is empty). It declares a sitemap at `https://www.kalugaqueen.id/sitemap_index.xml`.
- sitemap: `sitemap_index.xml` is present, accessible (HTTP 200), and generated by Yoast SEO. It lists `page-sitemap.xml` and `blog-sitemap.xml`.
- HTTPS/headers: The site uses HTTPS and HTTP/2. `alt-svc: h3=":443"` indicates HTTP/3 support. `content-security-policy: upgrade-insecure-requests` header is present. `x-litespeed-cache: hit` indicates active caching. `x-dns-prefetch-control: on` is enabled.
- Platform/version: The site is hosted on `hostinger` with `hpanel`. The CMS is `WordPress 7.0` and is powered by `PHP/8.3.30`.
- Plugins/theme/security: `Yoast SEO` and `LiteSpeed Cache` plugins are active. The `content-security-policy: upgrade-insecure-requests` header is a security measure. WordPress REST API links are present.
