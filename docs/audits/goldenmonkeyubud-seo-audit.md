# SEO Audit Report: goldenmonkeyubud.com

**Date:** 2026-06-10  
**Server:** hostinger-wp (PHP 8.1.34)  
**Upload Years:** 6 years | **Active:** 2026  
**Themes installed:** 6 (astra, goldenmonkey-canggu.zip, goldenmonkey-ubud, goldenmonkey-ubud-git + default WP themes)

---

## Overview

goldenmonkeyubud.com is a **301 redirect-only domain** pointing to `https://www.goldenmonkeybali.com/contact/`. It has a full WordPress installation but immediately redirects all traffic to the main Golden Monkey Bali contact page. LiteSpeed Cache is active. Minimal robots.txt exists with `Crawl-Delay: 20` and no sitemap. The site also has **W3 Total Cache** installed alongside LiteSpeed Cache, which is a significant concern.

## Issues

1. **Identical auth keys with goldenmonkeysanur.com (critical security):** All 8 auth key/salt constants are **identical** to those on goldenmonkeysanur.com. This is a severe cross-site cookie forgery vulnerability. Cookies generated by either site can authenticate users on the other.
2. **Dual caching plugins (LiteSpeed Cache + W3 Total Cache):** Running two full-page caching plugins simultaneously can cause cache corruption, serve stale content, and trigger conflicts. W3 Total Cache should be removed.
3. **Minimal robots.txt:** No sitemap reference, only a crawl-delay directive. For a redirect domain this is minor, but ideally it should block all crawling.
4. **Redundant WordPress installation:** A full WP install exists purely for a 301 redirect — unnecessary maintenance burden and security risk.
5. **Git artifacts on server:** The `goldenmonkey-ubud-git` theme directory suggests a git-based workflow, but leaving this on production is a cleanup and information-disclosure concern.
6. **Weak database password:** The MySQL password `m@2SC*:?` is short and contains limited entropy.
7. **No WP_AUTO_UPDATE_CORE config:** Not explicitly set, meaning security updates may not be automatic.
8. **Table prefix `gmu2021_`** is custom and good.

## Recommendations

- **Generate unique auth keys and salts** — these must be different from every other site. Highest priority fix.
- **Deactivate and remove W3 Total Cache** — LiteSpeed Cache is sufficient and already configured.
- Add `Disallow: /` to robots.txt to prevent search engines from crawling this redirect domain.
- Remove the `goldenmonkey-ubud-git` directory and any other development artifacts.
- Consider replacing the full WP installation with a simple `.htaccess` redirect if the domain serves no other purpose.
- Rotate the database password to a stronger value.
- Set `define('WP_AUTO_UPDATE_CORE', true);` for automatic security updates.
- Keep only the active theme + twentytwentyfive; delete the rest.
