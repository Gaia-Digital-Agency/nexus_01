# Technical Audit — ubudbeautycentre.com
**Server:** hostinger · **Audited:** 2026-06-11 · **Status:** live

## Verified signals
- HTTP/redirects: A 301 redirect chain is present, directing traffic to `https://www.ubudbeautycentre.com/`. The `x-redirect-by: WordPress` header indicates WordPress is handling the redirect.
- robots.txt: Present and contains `User-agent: *` and `Crawl-Delay: 20`.
- sitemap: `sitemap_index.xml` is present and valid, generated by Rank Math SEO Plugin, listing `post-sitemap.xml` and `page-sitemap.xml`. `sitemap.xml` returns a 404.
- HTTPS/headers: HTTPS is enforced via 301 redirect. `content-security-policy: upgrade-insecure-requests` header is present. `x-litespeed-cache` indicates server-side caching is active. `cache-control: no-store, no-cache, must-revalidate` and `expires: Thu, 19 Nov 1981 08:52:00 GMT` on the 200 response indicate no browser caching. `x-powered-by: PHP/8.2.30` is exposed.
- Platform/version: WordPress 7.0. PHP/8.2.30.
- Plugins/theme/security: Elementor 4.1.2 is active. Rank Math SEO Plugin is used for sitemaps. A `sln_booking_session` cookie is set, suggesting a booking plugin. `content-security-policy: upgrade-insecure-requests` is present. `x-powered-by: PHP/8.2.30` is exposed.
- Schema presence: JSON-LD schema is present.

## Top technical fixes (analysis only — NOT executed)
1.  Remove or significantly reduce the `Crawl-Delay: 20` directive in `robots.txt` to allow efficient crawling by search engines.
2.  Implement appropriate browser caching headers for static assets and potentially for the HTML document itself (if content is not highly dynamic) to improve repeat visit performance and Core Web Vitals.
3.  Add a relevant and descriptive `<h1>` tag to the homepage to improve on-page SEO and user experience.
4.  Ensure `sitemap_index.xml` is the only sitemap submitted to Google Search Console, and consider redirecting `sitemap.xml` to `sitemap_index.xml` or removing the 404.
5.  Consider suppressing the `x-powered-by` header to avoid exposing server-side technology details.
