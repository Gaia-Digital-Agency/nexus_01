# Technical Audit — cascadesbali.com
**Server:** ce01 · **Platform:** wp · **Audited:** 2026-06-11 · **Status:** live

## Verified signals
- HTTP/redirects: http://cascadesbali.com → 301 → https://cascadesbali.com → 301 → https://www.cascadesbali.com; final 200 at https://www.cascadesbali.com/. Canonical host is www over HTTPS. Two redirect hops (http→https, then non-www→www) — functional but not collapsed into a single hop.
- robots.txt: Present at https://www.cascadesbali.com/robots.txt; references Sitemap: https://www.cascadesbali.com/sitemap_index.xml. Contains malformed/disordered blocks — Crawl-delay lines appear BEFORE their `User-agent:` line (MJ12bot, sp_auditbot, Yandex), and a trailing `Disallow: /cdn-cgi/` sits under no agent. `User-agent: *` has empty Disallow (allow-all). Note: SemrushBot-SA is explicitly Allowed (Disallow blank) — this contradicts the prior audit which claimed SemrushBot was blocked.
- sitemap: https://www.cascadesbali.com/sitemap_index.xml → 200, valid XML, generated by Rank Math. Child sitemaps use absolute domain locs (post-, page-, category-), not IP. lastmod current (2026-06-11). Healthy.
- HTTPS/headers: Valid TLS, HTTP/1.1 200. Server: nginx/1.24.0 (Ubuntu) front (apache2 inactive). Present: X-Frame-Options: SAMEORIGIN, X-Content-Type-Options: nosniff. MISSING: Strict-Transport-Security (no HSTS), no Content-Security-Policy. No X-Robots-Tag noindex header (indexable).
- Platform/version: WordPress 7.0 (wp-includes/version.php), PHP 8.3.x. Custom table prefix `cr2021_` (not default wp_). WP_DEBUG=false. Auth keys are real/unique (not placeholders). DISALLOW_FILE_EDIT not defined.
- Plugins/theme: 23 plugins (16 active). Active theme `cascades-git` (custom); inactive sibling `cascades` present. SEO = Rank Math (active). Cache = LiteSpeed Cache 7.8.1 + LiteSpeed object-cache dropin active.
