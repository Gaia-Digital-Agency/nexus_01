# SEO Lite Audit: pegasus.com.au (Pegasus Media & Logistics)

**Date:** 10 June 2026  
**Hosting:** hostinger-wp | **Tier:** 3

## Overview

Pegasus Media & Logistics is a 14-year-old domain (uploads dating back to 2012) with a long history but currently near-zero organic presence. SEMRush data shows only 3 keywords and 0 organic traffic, which is consistent with the site having migrated to Wix while the WordPress install remains on the server.

## Critical Finding: Site Has Migrated to Wix

This WordPress installation on hostinger-wp appears to be **abandoned in favour of Wix**. The live `sitemap.xml` is generated by Wix (`generatedBy="WIX"`), and the Wix infrastructure presents a 404 error page for WordPress-native paths like `wp-sitemap.xml`. The WordPress install is still present on the server but the actual public-facing site is Wix-hosted.

## Robots.txt & Sitemap

- **robots.txt:** **Not found** — no robots.txt exists in the WordPress installation. However, since the site is now on Wix, this may be handled at the Wix level.
- **Sitemap:** `www.pegasus.com.au/sitemap.xml` returns a Wix-generated sitemap index with three sub-sitemaps:
  - `store-products-sitemap.xml`
  - `blog-posts-sitemap.xml`
  - `pages-sitemap.xml`
  The WordPress native `wp-sitemap.xml` returns a Wix 404 page.

## Key Issues

1. **WordPress install is effectively abandoned** — the public site runs on Wix. SEO efforts should focus on the Wix site, not this WordPress backend.
2. **No robots.txt in the WordPress install** — irrelevant since the WP site isn't serving traffic, but notable.
3. **Sydney theme needs update** (v2.57 → v2.68) if the WP install were active.
4. **SEMRush shows 3 keywords / 0 traffic** — consistent with a 14-year-old site that has effectively no organic search presence. The Wix migration may have reset rankings entirely.

## Recommendations

Given the site is on Wix:
- Focus SEO efforts on the Wix platform — configure Wix SEO tools, submit Wix sitemap to GSC, and set up Wix-native robots.txt.
- Clean up the WordPress install on hostinger-wp: either remove it to avoid confusion or at minimum disable it from serving anything.
- The 3 SEMRush keywords suggest there is minimal organic baseline to recover; a fresh SEO strategy for the Wix site is needed.
