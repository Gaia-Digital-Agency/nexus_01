# Technical Audit — institutescoffier.com
**Server:** hostinger · **Audited:** 2026-06-11 · **Status:** live

## Verified signals
- HTTP/redirects: All requests consistently redirect to `https://www.institutescoffier.com/` with a 301 status code. Redirects are handled by WordPress. HTTP/2 is in use, and HTTP/3 (h3) is supported.
- robots.txt: A standard WordPress `robots.txt` file is present, disallowing `/wp-admin/` but allowing `/wp-admin/admin-ajax.php`.
- sitemap: `sitemap_index.xml` is present and generated by Yoast SEO, listing several sitemaps (posts, pages, projects, contact form logs, testimonials, categories). However, `sitemap.xml` returns a 404 error. Some sitemap entries have `lastmod` dates from 2021.
- HTTPS/headers: The site serves over HTTPS. A `content-security-policy: upgrade-insecure-requests` header is present. The `x-powered-by: PHP/8.2.30` header is exposed. The `server: hcdn` header indicates a CDN is in use. Caching headers (`expires`, `cache-control`) are present on redirect responses.
- Platform/version: The CMS is WordPress, indicated by `wp-json` links and `x-redirect-by` header. The PHP version is 8.2.30. The hosting provider is Hostinger, using hPanel.
- Plugins/theme/security: Yoast SEO plugin is in use (from sitemap stylesheet). The `content-security-policy: upgrade-insecure-requests` header is set. The `generator` meta tag is not present. The `x-powered-by` header exposes the PHP version. JSON-LD schema is present.

## Technical findings
| Area | Finding | Severity |
|---|---|---|
| HTTP/redirects | Consistent 301 redirects to HTTPS www version. | Info |
| Sitemap | `sitemap.xml` returns a 404, while `sitemap_index.xml` is the correct index. This can lead to search engines failing to discover all content if they default to `sitemap.xml`. | High |
| Sitemap | Some sitemap entries (`project-sitemap.xml`, `project_category-sitemap.xml`) have `lastmod` dates from 2021, indicating potentially outdated content or sitemap generation issues. | Med |
| On-page | Homepage has no `<h1>` tag. | High |
| Security | `x-powered-by` header exposes PHP version (PHP/8.2.30). | Low |
| Speed/CWV | CDN (Hostinger CDN) is in use, with `x-hcdn-cache-status: HIT` for redirects and `DYNAMIC` for the final page. | Info |
| Speed/CWV | Upstream response times (`x-hcdn-upstream-rt`) are relatively high (1.481s and 2.534s). | Med |
