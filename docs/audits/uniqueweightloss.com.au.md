# Technical Audit — uniqueweightloss.com.au
**Server:** hostinger · **Audited:** 2026-06-11 · **Status:** live

## Verified signals
- HTTP/redirects: Non-HTTPS and non-www URLs (implied by the 301 redirect to HTTPS www) redirect with a 301 status to `https://www.uniqueweightloss.com.au/`. Redirects are handled by WordPress.
- robots.txt: Allows all user-agents (`Disallow:` is empty). Specifies sitemap location: `https://www.uniqueweightloss.com.au/sitemap_index.xml`.
- sitemap: `sitemap_index.xml` is found and lists `page-sitemap.xml`. `sitemap.xml` returns a 404 status. Sitemaps are generated by Yoast SEO.
- HTTPS/headers: Site serves over HTTPS using HTTP/2 protocol, with HTTP/3 (h3) support indicated. `content-security-policy: upgrade-insecure-requests`, `x-powered-by: PHP/8.3.30`, and `x-dns-prefetch-control: on` headers are present.
- Platform/version: CMS is WordPress 7.0. PHP version is 8.3.30.
- Plugins/theme/security: Yoast SEO and Elementor 4.1.1 plugins are active. LiteSpeed Cache is active. WordPress REST API endpoints (`wp-json`) are exposed. The `generator` meta tag exposes WordPress and Elementor versions.
- schema presence: JSON-LD schema is present on the homepage.
